- Anime with all spots at same location (e.g., school rooftop scenes)
- Single spot result → skip optimization, show simple itinerary
- Zero spots → clear error message
- >10 waypoints → Google Maps URL splits into parts
- Spots without coordinates → excluded from route, listed separately
